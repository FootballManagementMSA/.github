### 🔍 기획 배경

> 세종대학교 학사정보시스템 API를 활용하여 세종대학교 축구 동아리 관리 앱을 개발

실제 구글 플레이스토어에 배포하여 실용화될 수 있도록 개발

##### 앱의 기능

- 로그인 (이때, 로그인은 세종대학교 학사정보시스템 API를 활용하여 진행하도록 설정)

- 구단(교내 축구 동아리이지만 구단이라는 명칭을 사용) 생성

- 일정 생성(일정의 정보, 경기 날짜, 경기 장소, 스쿼드)
